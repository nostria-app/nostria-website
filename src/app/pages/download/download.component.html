Need help choosing?

Windows

Use the Microsoft Store for the simplest updates, EXE installers for direct installs, or MSI packages for managed deployments.

Linux

Choose AppImage if you want a portable binary, or .deb if you prefer package-managed installation on Debian-based systems.

Mac

The universal DMG works on both Intel Macs and Apple Silicon, so there is only one download to remember.
