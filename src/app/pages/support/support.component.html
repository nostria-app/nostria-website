Common Questions

How do I report a bug?

Visit our GitHub Issues page and create a new issue with the bug report template. Include as much detail as possible about the issue you're experiencing.

How can I request a new feature?

We love hearing feature ideas! Create a feature request on our GitHub Issues page or reach out to our support team on Nostr.

What's the best way to get quick help?

For quick questions and real-time support, contact us through our Nostria Support Account. Our team monitors this account regularly and can provide personalized assistance.
